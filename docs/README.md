# ~/fahadh

Personal documentation portal — a programmer-themed static site built with HTML, vanilla JavaScript, and Tailwind CSS.

## Pages

- **Home** — terminal-style hero with typing animation, doc list, tech stack badges
- **About** — bio with collapsible contact cards (Email, GitHub, Telegram, Twitter)
- **404** — curl-inspired error page

## Tech Stack

- HTML5 + vanilla JS
- [Tailwind CSS](https://tailwindcss.com) (CDN)
- [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + Inter fonts

## Running Locally

Open `index.html` directly or serve with any static server:

```bash
npx live-server
```

## Deployment

Deployed via GitHub Pages at [fahadh4ilyas.github.io](https://fahadh4ilyas.github.io).
